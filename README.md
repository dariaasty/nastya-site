# nastya-site

Стартовый репозиторий для сайта Насти.

## Что внутри

- `index.html` - основная страница
- `style.css` - стили и адаптив
- `script.js` - небольшое интерактивное поведение

## Как открыть

Просто открой `index.html` в браузере.

## Что можно сделать дальше

- заменить тексты на реальные
- добавить фотографии и проекты
- подключить форму, аналитику или сборщик
